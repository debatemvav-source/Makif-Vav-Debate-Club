🏆 מועדון דיבייט מקיף ו' - המדריך המלא לניהול האתר

ברוכים הבאים לאתר הרשמי והמשודרג של מועדון הדיבייט! האתר נבנה בטכנולוגיה מתקדמת המאפשרת שליטה מלאה בתוכן מבלי לדעת תכנות. כל הטקסטים מנוהלים דרך GitHub, וכל התמונות מנוהלות דרך Google Drive.

🖼️ ניהול תמונות (Google Drive)

האתר שואב את התמונות שלו באופן אוטומטי מתוך הדרייב של המועדון. בתיקייה הראשית ישנן 4 תתי-תיקיות שעליכם להכיר (חשוב לשמור על השמות באנגלית):

  • Logo - הכניסו לכאן את תמונת הלוגו של המועדון. האתר יזהה ויחליף את הלוגו למעלה.
  • Hero - הכניסו לכאן תמונה אחת (סלוגן/רקע) שתוצג בחלק העליון והראשון של האתר.
  • About - הכניסו לכאן תמונות שילוו את פסקאות חזון המועדון (הן יופיעו אוטומטית לצד הטקסטים בעמוד "אודות").
  • Gallery - כאן ניתן להכניס כמה תמונות שרק תרצו! האתר יציג אותן כמצגת מתחלפת ויפהפייה בפינת הגלריה.
שימו לב: אם תמחקו תמונה מהדרייב או תוסיפו אחת חדשה, היא תתעדכן באתר באופן מיידי כשמרעננים את העמוד!

📝 ניהול ועריכת טקסטים (GitHub)

כל מילה, כותרת, או תגובה באתר נשלטים דרך קובץ אחד שנמצא בגיטהאב ונקרא content.json.

איך עורכים?

  1. היכנסו לאתר GitHub ופתחו את הקובץ content.json.
  2. לחצו על סמל העיפרון (✏️) בצד ימין למעלה כדי להיכנס למצב עריכה.
  3. שנו רק את הטקסטים שנמצאים בתוך המרכאות הירוקות. אל תמחקו את המרכאות עצמן או פסיקים שבסוף השורה.
  4. בסיום, גללו מטה ולחצו על הכפתור הירוק Commit changes.

מבנה הקובץ ואיך לערוך כל עמוד:

1. אזור האודות (About)

האזור מחולק לפסקאות. לכל פסקה יש subtitle (כותרת משנה) ו-text (התוכן עצמו). ניתן לשנות אותם בקלות:

{ "subtitle": "קהילה שהיא מעבר למילים", "text": "כאן נכנס הטקסט המלא על המועדון שלנו..." }

2. פינת תגובות וחוויות (Testimonials)

כדי להוסיף חוויה של תלמיד חדש, פשוט הוסיפו name (שם התלמיד) ו-quote (הציטוט שלו):

{ "name": "דנה מ.", "quote": "החוויה הכי משמעותית שהייתה לי בבית הספר." }

3. שאלות ותשובות (FAQ)

כדי לשנות שאלות, חפשו את אזור ה-faq, שם מופיע q (השאלה) ו-a (התשובה):

{ "q": "האם חובה לדעת אנגלית?", "a": "לא, אנחנו לומדים יחד ומשתפרים." }

4. כותרות ויצירת קשר (General / Home)

בראש הקובץ תוכלו לשנות גם את המייל של המועדון, הקישור לאינסטגרם, וכותרות העמוד הראשי (כמו "הקול שלך שווה יותר").

💡 טיפ זהב לאחזקת האתר: תמיד כשאתם שומרים שינוי ב-GitHub (Commit changes), לוקח לשרתים של אינטרנט לעדכן את הקבצים, ולכן **יש להמתין 3 עד 5 דקות**. אם עבר זמן ואינכם רואים את השינוי, לחצו במקלדת על Ctrl + F5 כדי למחוק את זיכרון הדפדפן ולראות את הגרסה החדשה!
