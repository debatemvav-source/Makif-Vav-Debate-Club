🏆 מועדון דיבייט מקיף ו' - מדריך למנהל האתר (מפורט)

שלום למנהל האתר! מדריך זה נכתב בצורה מפורטת ושלב-אחר-שלב כדי להבטיח שתוכל לנהל את האתר בקלות, גם ללא שום ידע טכני מוקדם.

חלק א': החלפת תמונות (דרך Google Drive)

האתר תוכנת כך שישאב תמונות ישירות מהגוגל דרייב. זה אומר שלא צריך לגעת בקוד כדי להחליף תמונה!

מבנה התיקיות החובה:

בתוך תיקיית האתר בדרייב, יש בדיוק 4 תתי-תיקיות. אסור לשנות את השם שלהן באנגלית.

  1. Logo - תיקיית הלוגו. הכניסו לכאן תמונה אחת בלבד. התמונה הזו תופיע בקטן בפינת האתר למעלה.
  2. Hero - תיקיית הרקע הראשי. הכניסו לכאן תמונה אחת בלבד (רצוי תמונה לרוחב, באיכות טובה). זו התמונה הענקית שרואים מיד כשנכנסים לאתר מאחורי משפט הסלוגן.
  3. About - תיקיית ה"אודות". כאן מכניסים בדרך כלל 3 תמונות. התמונות האלו יופיעו אוטומטית ליד פסקאות ההסבר על חזון המועדון.
  4. Gallery - תיקיית הגלריה. לכאן אפשר לזרוק עשרות תמונות! המערכת תיקח את כל התמונות שבתיקייה הזו ותיצור מהן מצגת מתחלפת בתחתית האתר.
איך לרענן תמונות? ברגע שהוספת או מחקת תמונה בגוגל דרייב, השינוי חל מיד! אם נכנסת לאתר והתמונה הישנה עדיין מופיעה, זה קורה בגלל הזיכרון של הדפדפן שלך. פשוט לחץ במקלדת על Ctrl + F5 (זה נקרא "רענון קשיח") ותראה את השינוי.

חלק ב': עריכת טקסטים (דרך GitHub)

כל מילה שמופיעה באתר (החל מסלוגן ועד לציטוט של תלמיד) שמורה בקובץ מילון פשוט בשם content.json.

צעד אחר צעד לאיך עורכים:

  1. היכנסו לאתר GitHub.com והתחברו למשתמש של הדיבייט.
  2. במסך הראשי, היכנסו לפרויקט של האתר (debate-club-website או דומה).
  3. רשימת הקבצים תופיע מולכם. לחצו על הקובץ content.json.
  4. בצד ימין למעלה של המסך, תראו סמל של עיפרון (✏️). לחצו עליו כדי להיכנס למצב עריכת טקסט.
⚠️ כלל ברזל לעריכת JSON (חשוב מאוד!):
הקובץ בנוי בזוגות של "מפתח": "טקסט שלכם".
1. חובה לשמור על המרכאות הירוקות (") בתחילת ובסוף הטקסט שלכם.
2. חובה לשמור על הפסיק (,) בסוף כל שורה (אם היה שם אחד).
מחיקה בטעות של מרכאות או פסיק תגרום לאתר להפסיק לעבוד עד שיתקנו את זה.

המדריך המלא לאזורים בקובץ content.json:

1. עריכת כותרת ופרטי קשר (General / Home)

תחת האזור של general או home, תמצאו את המשפטים הראשיים:

"heroTitle": "הקול שלך שווה יותר.", "heroSubtitle": "הצטרפו למועדון הדיבייט של מקיף ו' ותלמדו לדבר, לשכנע ולהוביל.", "email": "debatemvav@gmail.com"

כדי לשנות את המשפט המרכזי, פשוט שנו את הקול שלך שווה יותר לכל דבר אחר, אבל הקפידו לא למחוק את המרכאות שעוטפות אותו.

2. עריכת עמוד האודות (About)

אזור האודות מחולק ל"פסקאות" (Paragraphs). כל פסקה כוללת subtitle (כותרת משנה מודגשת) ו-text (פירוט נרחב). תוכלו לשנות את התוכן שלהן כך:

{ "subtitle": "כאן כותבים את כותרת המשנה", "text": "כאן מכניסים את פסקת ההסבר הארוכה שמתחתיה." }

3. פינת חברי המועדון ממליצים (Testimonials)

כדי לשנות את שמות התלמידים הממליצים או את מה שהם אומרים, רדו לאזור ה-testimonials. תראו שם רשימה של חברים. לכל חבר יש name (שם החבר) ו-quote (ציטוט/המלצה):

{ "name": "דנה מ.", "quote": "לפני הדיבייט בקושי הצלחתי להצביע בכיתה. היום אני יכולה לעמוד מול אולם שלם." }

4. עריכת שאלות נפוצות (FAQ)

אזור השאלות בנוי משאלות (q = Question) ותשובות (a = Answer):

{ "q": "מתי מתקיימים המפגשים?", "a": "בימי שלישי בשעה 14:00 בספרייה." }

איך שומרים את השינויים בסוף העריכה?

  1. לאחר שסיימתם לערוך ולבדוק שלא מחקתם בטעות מרכאות, רדו לתחתית הדף לחלוטין.
  2. תראו כפתור ירוק בשם Commit changes (או כפתור ירוק דומה למעלה מימין). לחצו עליו.
  3. בחלון הקטן שיקפוץ, לחצו שוב על Commit changes (הכפתור הירוק).
⏳ טיפ אחרון וחשוב מאין כמותו: הזמן שגיטהאב לוקח!
שמרתם שינוי טקסט? מצוין. שרתי גיטהאב נמצאים בארצות הברית ולוקח להם "להפיץ" את האתר שלכם מחדש. זה לוקח בין 3 ל-5 דקות.
אל תלחצו אם הטקסט לא משתנה מיד. המתינו 4 דקות, פתחו את האתר במחשב ולחצו Ctrl + F5. אם אתם בטלפון, היכנסו לאתר במצב "גלישה בסתר" (Incognito) או מחקו את היסטוריית הגלישה כדי לא לראות את האתר הישן ששמור בזיכרון של הטלפון.
